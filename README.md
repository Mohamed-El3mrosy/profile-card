# 📌 Profile Card Project

## 📖 Overview

The **Profile Card** project is a simple and responsive web component that displays user information in a clean and modern card layout. It focuses on layout design, styling, and responsiveness using modern CSS techniques.

This project is ideal for practicing front-end fundamentals and improving UI design skills.

---

## 🚀 Technologies Used

* **HTML5** – for structuring the content
* **CSS3** – for styling and layout
* **Flexbox** – for flexible and responsive alignment
* **CSS Grid** – for advanced layout control

---

## ✨ Features

* 🎨 Clean and modern UI design
* 📱 Fully responsive across different screen sizes
* ⚡ Optimized layout using Flexbox and Grid
* 🖼️ Profile image, name, and additional user info display

## 🛠️ Installation & Usage

1. Clone the repository:

   ```bash
   git clone 	https://github.com/Mohamed-El3mrosy/profile-card
   ```

2. Navigate to the project folder:

   ```bash
   cd profile-card
   ```

3. Open `index.html` in your browser.
